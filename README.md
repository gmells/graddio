# TRNC HOME SPOTTER SYSTEM

  _This system aims to create a real estate web application for the TRNC(Turkish Republic of  Northern Cyprus)_

# TOOLS AND FRAMEWORKS
- React.js
- Javascript
- Tailwind css
- Vite
- MongoDB
- Express.js
- Node.js
- Firebase
- Insomnia
